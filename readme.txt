For running the program:

1- (Recommended) Create a virtualEnv : Open the vscode terminal and write (on Windows):  python -m venv yourVenvName

2- Activate your new virtual env , open the terminal and write : yourVenvName/scripts/activate 

3- Install this libraries : 

    Open the terminal and write:

     pip install torch
     pip install speechrecognition
     pip install pyaudio
     pip install numpy
     pip install nltk    
     pip install pyttsx3

4- In the vscode terminal write : python .py

5- Enjoy!
